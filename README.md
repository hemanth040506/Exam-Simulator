# Exam Simulator

A full-stack online examination simulator supporting multiple exams like
GATE, JEE (Main), and EAMCET.

## Features
- Exam-wise sets (date & shift based)
- Guidelines before exam start
- MCQ, MSQ, and Integer/NAT question support
- Backend-driven evaluation
- Secure authentication (JWT)
- MongoDB-based question storage

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB Atlas
- Auth: JWT
